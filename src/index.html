Інструкція по поклейці наліпок

Помити та знежирити поверхню

Помийте машину без воску, або знежирте поверхню для поклейки засобом для миття вікон або спиртом.

Карткою розгладити транспортну плівку

Візьміть пластикову картку, покладіть наліпку на тверду рівну поверхню. Прогладьте наліпку карткою по монтажній плівці для кращого сцеплення елементів.

Зняти частину підложки

Переверніть наліпку та зніміть та замніть 2-3 см підкладки (бумажна частина на якій розташовані елементи)

Поклейка наліпки

Прикладіть наліпку до поверхні, розгладьте карткою по відкритій частині монтажної плівки. Знімайте по маленьку підкладку та рівномірно розглажуйте наліпку.
